Live Coding with Fluxus and TidalCycles

This ten-minute screen recording, made in February 2023, demonstrates live coding with Fluxus and TidalCycles. In this video several pieces of free software are working in tandem with one another. Fluxus and TidalCycles are running as Emacs processes. In this regard Emacs is interfacing with TidalCycles and Fluxus. Code written live is evaluated and sent to Fluxus and TidalCycles. The audio furthermore influences the visuals. Behind the scenes the JACK audio server is making this possible. JACK is transforming a stereo audio signal from TidalCycles into a mono signal and sending this to Fluxus. Open Broadcaster Software (OBS) was responsible for recording sonic and visual elements of the performance.
