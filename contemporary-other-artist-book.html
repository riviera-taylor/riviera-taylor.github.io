Contemporary Other Artist Book

I designed the Artist Book for the Contemporary Other showcase which took place at Ugly Duck in London. I created the publication using Emacs, Pandoc, ConTeXt and Scribus. All of these are pieces of Free / Libre / Open Source Software. From the outset I envisioned using full-page images for a strong visual effect. The process was collaborative insofar others contributed copy to the publication. I collated this copy and pieced it together in one document with Emacs Org Mode. I used Pandoc to export the copy to an encoded format comprehensible to ConTeXt. Next, I set about designing the layout of the artist book, adding in photographs and layout text using ConTeXt macros. These decisions were less collaborative and more autonomous. I determined the order of different sections with printing in mind.
